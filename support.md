# AudioThrough Support

**AudioThrough** is a macOS menu bar app for routing audio from one input source to multiple output destinations.

## Contact

For support, bug reports, or general questions about AudioThrough:

**Email:** [drewnicholsmusic@gmail.com](mailto:drewnicholsmusic@gmail.com)

Please include your macOS version and a short description of the issue. Screenshots or steps to reproduce are helpful.

## Before you write

- **No audio?** Check that an input is selected in settings, the configuration is enabled, at least one mapping has an output device and is enabled, and microphone access is granted for AudioThrough in **System Settings → Privacy & Security → Microphone**.
- **Setup help:** See the in-app user guide under **Help → User Guide**.
